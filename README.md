# PPTtimer

#### 介绍
基于Qt开发的一个PPT演讲倒计时软件。

开发这个小软件的契机是出于一次偶然：在讲PPT的时候，我演讲的时间超过了规定时间。导致后面有几页PPT没有讲，虽然影响不大，但是我觉的这样不好！于是，开发了这款小软件,希望你能够喜欢！欢迎大家优化！issue
